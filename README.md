# Simple Bot for Bot Framework Emulator

This project is a traditional chatbot built with Python and tested in Bot Framework Emulator.

## Features
- Responds to multiple prompts
- Shows capabilities
- Handles invalid input
- Works in Bot Framework Emulator
- Easy to extend later

## Commands
- hello
- help
- hours
- services
- contact
- bye

## Run steps
1. Install requirements
2. Run `python app.py`
3. Open Bot Framework Emulator
4. Connect to `http://localhost:3978/api/messages`
